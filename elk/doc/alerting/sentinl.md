# Sentinl
### 安装：
1. 进入到kibana安装目录下：
```
$ cd ${home}/kibana/bin
```
2. 执行安装命令：

```
$ ./kibana-plugin install https://github.com/sirensolutions/sentinl/releases/download/tag-6.2.3-3/sentinl-v6.2.4.zip
```

3. 启动kibana：
```
$ nohup ./kibana & 
```
